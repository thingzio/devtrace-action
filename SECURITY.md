# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v1.x    | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue.**

Instead, use [GitHub's private vulnerability reporting](https://github.com/thingzio/devtrace-action/security/advisories/new) to submit your report. You will receive a response within 72 hours acknowledging receipt.

## Scope

This policy covers the `devtrace-action` GitHub Action code. Issues related to the DevTrace API service itself should be reported to [security@thingz.io](mailto:security@thingz.io).
